Ysph-thesis

MPH Thesis comparing deterministic and stochastic modeling methods when applied to school closings to prevent influenza transmission

Download this project as a .zip file Download this project as a tar.gz file

Abstract

Specific Aims: Designing a model that informs policy makers when to close schools in order to prevent further flu transmission.

Hypotheses: Demonstrating that stochastic models are better are a better fit to predict possible flu epidemics in elementary schools than deterministic models.
